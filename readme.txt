== Description ==
The SM Login Styler is a wordpress plugin to modify your wordpress admin login or registrestion page.

= Features =
* Settings Page to set Login Page options.
* You can change login page logo link url and title.
* You can set login page background image.
* Can set  form color and lable color from here.
* Also you can set logo size from here.

== Installation ==
1. You can install the SM News Ticker form wordpress.org plugin or download this plugin and extract is to your wordpress site plugin folder.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now SM Login Styler Plugin activate to your wordpress site.

= Usage =

1. Go to SM Login Styler form left navigation menu.
2. Set all the option for your Login page
3. Hit the save button.
